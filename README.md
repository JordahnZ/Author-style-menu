# Author-style-menu

Transform any writer's voice into actionable writing tools.

## Testing a production build

This project does not yet include a build system, but if it did you would typically:

1. Install dependencies: `npm install`.
2. Create a production build: `npm run build`.
3. Run tests against the build output: `npm test`.

These commands assume an npm-based workflow and would need to be adapted for other environments.
